# TypeScript 学习路线图

> 本路线图专为具有 **C++ / Python** 开发经验的工程师设计。
> 跳过零基础内容，聚焦 TypeScript 及其生态系统的核心知识，
> 帮助你在 **33–48 天**内建立完整的 TypeScript 技术栈能力。

---

| 阶段 | 标题                  | 预计时间   | 前置要求            |
|------|-----------------------|-----------|---------------------|
| 01   | JavaScript 基础       | 3–5 天    | C++/Python 经验     |

## 快速开始指南

1. **评估起点** — 如果你熟悉 JavaScript，可直接从 Stage 02 开始
2. **按依赖顺序推进** — 参考上方依赖图，确保前置阶段已完成
3. **Stage 07 可并行** — 四个子阶段相互独立，按兴趣选择顺序
4. **每阶段包含** — 核心概念、代码示例、练习题、参考资源
5. **建议节奏** — 每天 2–4 小时，保持持续学习

> 💡 **提示：** 具有 C++ 模板元编程经验的开发者，TypeScript 的类型体操会感觉很熟悉；
> 熟悉 Python 的开发者则会在异步编程（Stage 04）中找到共鸣。
